# Indian E-Commerce Customer Behavior Analysis (SQL)

## Project Overview
This project analyzes an Indian E-Commerce Customer Behavior dataset using MySQL.

## Dataset
- Indian E-Commerce Customer Behavior Dataset

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- Aggregate Functions
- CASE WHEN
- Window Functions
- CTE

## Business Questions Solved
1. Total Revenue
2. Average Revenue
3. Revenue by Product Category
4. Revenue by Marketing Channel
5. Top Customers
6. Cart Abandonment Analysis
7. Purchase Rate by Device Type
8. Running Total Revenue
9. Customer Ranking
10. Revenue by Payment Method
